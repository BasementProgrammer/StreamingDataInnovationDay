# StreamingDataInnovationDay
Code base for the Sreaming Data AI Innovation Day

/md                     - Markdown documentation files
/tf                     - Terraform files
/testclient             - Streaming data test client to send data to the built application.


In this Streaming Data Innovation day you will learn how to create a working solution that accepts streaming data in from endpoints in the field, stored theat data in an Oracle Autonomous AI Database, Leverages AI built in the database to detect anomolies in the data, and then uses Oracles AI based visualization capeabilities to render insights into that data.
